アプリ詳細

このアプリはあらゆる端末を悪意のあるユーザから守ります。

このアプリは、プライバシーとオンラインの自由のための世界最強のツールの開発者である Security Guardプロジェクトによってサポートされている唯一の公式モバイル ブラウザです。

Security Guardは常に無料ですが、寄付によって可能になります。

Security Guardプロジェクトは、日本国を拠点とする非営利団体です。

今日の貢献。すべてのギフトが違いを生みます。

ブロックトラッカー

Security Guardは、アクセスする各 Web サイトを分離するため、サードパーティのトラッカーや広告があなたを追跡できません。

閲覧が終了すると、Cookie は自動的に消去されます。

監視から守る

Security Guardは、あなたの接続を監視している誰かが、あなたがアクセスした Web サイトを知ることを防ぎます。

あなたのブラウジング習慣を監視している人は、Security Guardを使用していることしかわかりません。

指紋防止

Security Guardは、すべてのユーザーが同じように見えるようにすることを目的としており、ブラウザーとデバイスの情報に基づいてフィンガープリントを取得することを困難にします。

多層暗号化

Security Guardを使用すると、トラフィックは Security Guard ネットワークを通過する際に 3 回中継され、暗号化されます。

このネットワークは、Security Guard リレーと呼ばれるボランティアが運営する何千ものサーバーで構成されています。

仕組みの詳細については、このアニメーションをご覧ください。

自由に閲覧

Security Guardを使用すると、ローカルのインターネット サービス プロバイダーがブロックしている可能性のあるサイトに自由にアクセスできます。

このアプリは、あなたのような寄付者によって可能になりました Security Guardは、非営利団体である Security Guard Project が開発した無料のオープン ソース ソフトウェアです。寄付を行うことで、Security Guard を強力で安全で独立したものに保つことができます

Security Guardプロジェクトについて

Security Guard Project, Inc. は、人々を追跡、監視、および検閲から保護し、オンラインでのプライバシーと自由のための無料およびオープン ソース ソフトウェアを開発しています。

Security Guard プロジェクトの使命は、無料でオープンソースの匿名性とプライバシー技術を作成して展開し、それらの無制限の可用性と使用をサポートし、科学的および一般的な理解を促進することにより、人権と自由を促進することです。
